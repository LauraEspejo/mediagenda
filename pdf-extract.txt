TALLER MODELO DE CASO  
 
SISTEMA WEB DE GESTIÓN DE CITAS MÉDICAS PARA UNA CLÍNICA  
Objetivo:  Desarrollar una aplicación web funcional que permita a pacientes 
agendar citas y a la clínica gestionarlas, aplicando conceptos de desarrollo 
moderno, calidad del software y seguridad web.  
1. Contexto del caso  
Una clínica necesita digitalizar su proceso de asignación de citas, ya que 
actualmente se realiza de forma manual, generando errores, demoras y pérdida 
de información. El sistema debe permitir registro de usuarios, agendamiento, 
consulta de citas y administración interna.
